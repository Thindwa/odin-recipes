# Recipe Website

## Description
This project is a simple recipe website created to practice basic web page structure and content organization. It includes multiple recipe pages, each with a description, a list of ingredients, and step-by-step instructions. The website demonstrates clear headings, lists, and images to make the recipes easy to follow.

## Features
- Index page linking to all recipes  
- Individual recipe pages for each dish  
- Free images of finished dishes  
- Structured sections for description, ingredients, and steps  

## Pages Included
- **Index Page** – lists all recipes with clickable links  
- **Spaghetti Bolognese** – classic pasta recipe  
- **Chicken Stir Fry** – quick and healthy stir fry recipe  
- **Banana Pancakes** – simple breakfast/snack recipe  

## How to Use
1. Open the `index.html` page in a web browser.  
2. Click on any recipe link to view the full recipe.  
3. Read the description, check the ingredients, and follow the steps to cook the dish.  

## Purpose of the Project
The purpose of this project is to demonstrate understanding of:  
- Web page structure (headings, paragraphs, lists)  
- Organizing content in a readable format  
- Linking multiple pages together  
- Adding images to enhance content
